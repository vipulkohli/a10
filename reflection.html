Java Text/Image File Reading and Reformatting:
TXT to CSV conversion:
In order to insert some of the test questions to validate the tutors in Math, Science or History, we had to read, discern, and reformat test questions from online released Texas Assessment of Knowledge and Skills exams. The Java program and its user convert these online questions to Snap2Ask's standard comma-delimited (CSV) format that is used to insert questions to the database through the following steps (input and output shown in Figure 1.1):
1. The user copies and pastes desired questions to one .txt file which may include unnecessary text such as test-taking directions
2. The user copies and pastes the answer key to another .txt file with answers choices A, B, C, D or F, G, H, J
3. The Java program reads both files and converts deletes an answer choice to meet the Snap2Ask test standard of three answer choices, A to C, not four as in the TAKS tests.
4. The Java Program associates an answer and a subject with the question and prints out the CSV results.

5Which of these could be used to determine whether a substance is a base?

Which of these could be used to determine whether a substance is a base?, Thermometer, Bunsen burner, Balance Litmus paper,C,Science

public static void writeFilesToFolder(final File inFolder) {
        try{
                int k = 0;
                for (final File fileEntry : inFolder.listFiles()) {
                    k++;
                    System.out.println(fileEntry.getName());
                    String loc = "realpng/" + fileEntry.getName();
                    File file = new File(loc);
                    BufferedImage image = ImageIO.read(file);
                    image = fillTransparentPixels(image, Color.WHITE);
                    File f = new File("indexed_test" + k + ".jpg");
                    ImageIO.write(image, "jpg", f);
            }
        }
        catch(Exception e){}
    }//END method
        public static BufferedImage fillTransparentPixels(BufferedImage image, Color fillColor) {
                int w = image.getWidth();
                int h = image.getHeight();
                BufferedImage image2 = new BufferedImage(w, h, BufferedImage.TYPE_INT_RGB);
                Graphics2D g = image2.createGraphics();
                g.setColor(fillColor);
                g.fillRect(0,0,w,h);
                g.drawRenderedImage(image, null);
                g.dispose();
                return image2;
            }
